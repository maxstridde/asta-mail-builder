How to send the mail in Thunderbird
  1. Compose a new mail, click into the mail body and delete everything until it shows normal text in the rich-text editor.
  2. Click Insert → HTML, paste the copied code and hit Insert.
  3. Go into the empty last row below the pasted HTML, press Enter once, then Delete twice to remove the unwanted last line.
  4. If that didn't work, redo the process following these exact steps.
